# URL과 URI

- Uniform Resource Identifier
- Uniform Resource Locator

URL은 URI의 하위 개념이다.

URI의 문법은 다음과 같다.

`scheme:[//[user:password@]host[:port]][/]path[?query][#fragment]`

web에서는 `scheme`에 프로토콜을 넣는다.

HTTP 1.1에서는 ASCII 문자만을 사용한다.
